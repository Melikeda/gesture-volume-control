# Gesture Volume Control 🎛️🖐️

Gerçek zamanlı el hareketleri ile bilgisayar ses seviyesini kontrol eden Computer Vision projesi.

Bu proje OpenCV, MediaPipe ve pycaw kullanılarak geliştirilmiştir. Başparmak ve işaret parmağı arasındaki mesafe analiz edilerek gerçek Windows ses sistemi kontrol edilmektedir.

---

# 🚀 Demo

Parmakları Aç  → Ses Artar  
Parmakları Kapat → Ses Azalır

Gerçek zamanlı gesture interaction sistemi sayesinde kullanıcı fiziksel temas olmadan bilgisayar sesini kontrol edebilir.

---

# 🧠 Kullanılan Teknolojiler

- Python
- OpenCV
- MediaPipe
- NumPy
- pycaw
- Windows Audio API

---

# ✨ Özellikler

✅ Real-time hand tracking  
✅ Gesture-based volume control  
✅ Dynamic volume bar UI  
✅ LOW / MEDIUM / HIGH states  
✅ Gesture smoothing system  
✅ Clipping system  
✅ Windows audio integration  
✅ Debug monitoring  
✅ Error handling  
✅ Safety systems
